# Mermaid_FlowChart Workflow

사용자가 흐름도/순서도/프로세스를 요청하면 즉각 실행되어 ELK 렌더러 기반 무결점 Mermaid 다이어그램 코드를 생성합니다.

## 실행 로직

1. **의도 파악 및 구조화**: 제공된 텍스트/문서에서 핵심 로직 추출
2. **레이아웃 선정**: 복잡한 노드(5개 이상)는 `defaultRenderer: "elk"` 1순위 채택
3. **Draft 작성 및 자가 검증**: 노드 ID 중복, 연결 오류 린팅
4. **결과 출력**: HTML 태그, 괄호 꼬임 제거한 순수 마크다운 코드 블록

## 출력 형식

```markdown
​```mermaid
%%{init: {"flowchart": {"defaultRenderer": "elk"}} }%%
flowchart TD
    A[시작] --> B{판단}
    B -->|Yes| C[처리]
    B -->|No| D[종료]
​```
```

## 사용 규칙

- **노드 ID**: 영문/숫자만 (한글 ID 금지)
- **레이블**: 한글 허용, 단 특수문자(`[]{}()`) 내부에서는 이스케이프 처리
- **화살표**: `-->` 기본, 조건부는 `-->|조건|`
- **Warning 노드**: 오류/주의 분기는 `:::warning` 클래스 사용

## 트리거 예시

- "이 프로세스 흐름도로 그려줘"
- "PDCA 사이클 순서도 만들어줘"
- "에러 복구 로직 다이어그램"
