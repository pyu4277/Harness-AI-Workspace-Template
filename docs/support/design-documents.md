# 설계문서 카탈로그 (28종)

> 계획/설계 단계 진입 시 이 목록에서 프로젝트에 필요한 문서만 선별한다.
> pdca plan, pdca design, plan-plus, harness-architect 등 어떤 경로든 적용.

---

## 선택 원칙

1. **전수 생성 금지** -- 28종을 전부 만들지 않는다. 프로젝트에 필요한 것만 선별.
2. **조건 기반 선택** -- 아래 조건 컬럼으로 해당 여부를 판단.
3. **사용자 확인** -- 선별 결과를 사용자에게 보여주고 추가/제거 승인 후 진행.

| 3 | 아키텍처 설계서 | **항상** | 시스템 구조 |

| 20 | 산출물 적용 계획표 | 팀 >= 2명 + 규모 >= Dynamic | 산출물 일정/담당자 |

| 22 | 메뉴 구성도 | UI 존재 시 (웹/모바일) | 화면 메뉴 구조 |

## 규모 판단 기준

| 규모 | 기준 |
|:---|:---|
| Starter | 정적 웹, CLI 도구, 단일 스크립트, 소규모 자동화 |
| Dynamic | 풀스택 앱, API 서버, DB 사용, 중규모 프로젝트 |
| Enterprise | 마이크로서비스, 대규모 팀(6명+), 복잡한 인프라 |
