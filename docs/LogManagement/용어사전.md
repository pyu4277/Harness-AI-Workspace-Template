# 용어사전

> 이 파일은 대화에서 등장한 전문용어를 모아두는 누적 사전입니다.
> AI에게 **"전문용어 정리해줘"** 라고 말하면 이 파일이 자동으로 업데이트됩니다.
> **"전문용어 알려줘"** 라고 하면 이 파일 경로를 안내합니다.

## 컬럼 설명

| 컬럼 | 의미 |
| :--- | :--- |
| **번호** | 행 순서 (참조 번호) |

| **원어** | 영어 또는 원래 언어 표기 |

| **비고** | 주의사항, 관련 용어, 출처 등 |
| **등록일** | 이 용어가 처음 추가된 날짜 |
