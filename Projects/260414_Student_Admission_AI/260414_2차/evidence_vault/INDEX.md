# Evidence Vault (2차) -- 참조 전용

> **주의**: 2차에서는 자체 evidence_vault 를 사용하지 않는다.
> 학생 객관 자료는 **`../260414_1차/evidence_vault/`** 를 참조한다.
>
> 결정 근거: 학생의 생활기록부/성적/자격증 등은 시간 불변 자산. 회차마다 복제할 이유 없음. (사용자 결정 2026-04-14)

## 참조 규약

- 자소서 인용 마커는 `[^ev_<id>]` 유지. 단, 실제 파일은 1차 vault 에서 탐색
- 2차 writer 는 1차 `evidence_vault/INDEX.md` 를 1순위로 참조
- 2차 자체 vault 하위 폴더는 유지하되 실파일은 넣지 않음 (구조 불변성 / 구조 검증용)
- 2차 INDEX.md 수정 시 post_tool_validate.sh 경고는 무시 (이 문서는 매핑 안내)
