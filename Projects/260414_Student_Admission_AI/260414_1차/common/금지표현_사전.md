# 금지표현 사전

post_tool_validate.sh 훅이 참조. 자소서/이력서/포트폴리오에서 감지 시 경고.

## 1. 과장 (근거 없이 사용 시 금지)

| 표현 | 허용 조건 |
|---|---|
| 최고의, 최상의, 최대의 | 공식 수치/공시 인용 시만 |
| 업계 1위, 역대급 | 기관 공식 발표 인용 시만 |
| 완벽한, 완전한 | 금지 (인간 활동에 완벽 없음) |
| 독보적인, 타의 추종을 불허 | 금지 |

## 2. 절대 단언

| 표현 | 이유 |
|---|---|
| 모든 ~ | 반례 가능성 |
| 항상, 언제나 | 예외 반박 여지 |
| 절대로, 결코 | 극단적 단언 |
| 100%, 무조건 | 증명 불가 |

## 3. 자기비하

| 표현 | 대안 |
|---|---|
| ~에 불과 | 담담히 서술 |
| 감히 | 삭제 |
| 외람되지만 | 삭제 |
| 미천한 | 삭제 |
| 부족하지만 | "지금부터 성장하겠습니다" |

## 4. 진부 표현 (단독 사용 금지 — 구체 증거와 함께만)

- 열정
- 최선
- 노력
- 성실
- 책임감
- 커뮤니케이션 능력
- 팀워크

→ 반드시 **증명 가능한 사례(evidence_vault 링크) + 정량 지표**와 함께 사용.

## 5. 자소서 금기 상투어

- "어릴 때부터 ~을 꿈꿔왔습니다" (진부)
- "저희 부모님은 ~" (지원자 본인에 집중)
- "오래전 일이지만" (구체 시기 명시)
- "작지만 큰 성과" (모순 표현)

## 6. 맥락별 주의 표현

| 맥락 | 피할 표현 |
|---|---|
| 성장배경 | "평범한 가정에서", "남들과 다르지 않은" |
| 지원동기 | "귀사의 ~에 끌려" (감상적) |
| 입사후포부 | "열심히 하겠습니다" (무의미) |
| 협업 경험 | "혼자 해냈다" (팀 기여 무시) |
